When the system receives an official HTML regulation page, a scanned PDF amendment, and a ministry guideline, it first classifies each source by authority. The official HTML regulation is Tier 1 if it comes from an official legislation or gazette domain. The scanned amendment is Tier 2 if it has legal force as an amendment or implementing regulation. The ministry guideline is Tier 3.

Next, the extractor uses BeautifulSoup or Playwright-derived HTML text for the Tier 1 page. For the scanned PDF amendment, it attempts Docling first if the PDF has embedded text, then Surya OCR, then Tesseract. Low-confidence or malformed OCR chunks are flagged for human review.

The Tier 3 guideline is never scored. It can only help locate article numbers or keywords that should be checked in Tier 1 or Tier 2 sources.

If Tier 1 and Tier 2 conflict on the same article, both versions are logged in the SQLite audit trail. The system scores from Tier 1, flags the conflict for human review, and preserves the Tier 2 text as supplementary context. The returned citation points to the Tier 1 source with the verbatim quote, article reference, URL, and document hash.

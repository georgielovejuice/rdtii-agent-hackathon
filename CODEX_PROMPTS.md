## ARCHITECTURE CONTEXT (read before any prompt)

The pipeline has TWO operating modes:

  Mode A — Full LLM (with API key)
    OpenAI gpt-4o or Anthropic Claude Sonnet 4
    ChromaDB semantic retrieval + NetworkX knowledge graph
    Span verifier (entity grounding + relation preservation)

  Mode B — Offline/Demo (no API key)
    Local heuristic classifier (deterministic, exact phrase match)
    Keyword retrieval fallback (no ChromaDB needed)
    Bundled legal text (no network needed)

Both modes enforce: No Quote → No Score. Uncertain → Human review queue.

LLM backend is selected by environment variable:
  LLM_BACKEND=openai      → OpenAI SDK (OPENAI_API_KEY)
  LLM_BACKEND=anthropic   → Anthropic SDK (ANTHROPIC_API_KEY)
  (neither set)           → heuristic fallback

  def _call_ollama(prompt: str) -> str | None:
      """
      Call a local Ollama server via its OpenAI-compatible API.
      Ollama must be running: `ollama serve`
      Model must be pulled: `ollama pull llama3.1:8b`
      """
      try:
          from openai import OpenAI
      except ModuleNotFoundError:
          logger.warning("[reason] openai package not installed — falling back to heuristic")
          return None

      base_url  = os.getenv("OLLAMA_BASE_URL", "http://localhost:11434/v1")
      model     = os.getenv("LLM_MODEL", "llama3.1:8b")

      try:
          client = OpenAI(
              api_key="ollama",          # Ollama does not check the key value
              base_url=base_url,
          )
          response = client.chat.completions.create(
              model=model,
              max_tokens=512,
              messages=[{"role": "user", "content": prompt}],
              temperature=0.0,           # deterministic — critical for legal scoring
          )
          return response.choices[0].message.content.strip()

          logger.info("[reason] Is Ollama running? Try: ollama serve")
